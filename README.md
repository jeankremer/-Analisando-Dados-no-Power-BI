# -Analisando-Dados-no-Power-BI
Análise de dados de finanças

Realizei uma análise detalhada dos dados de vendas e lucros de produtos por países. Durante o processo, examinei minuciosamente os registros de vendas, identificando as tendências e padrões de desempenho em diferentes regiões. Por meio dessa análise, pude compreender melhor o impacto das vendas em cada país, bem como o retorno financeiro obtido. Essas informações forneceram insights valiosos para aprimorar estratégias de mercado, identificar oportunidades de crescimento e otimizar a alocação de recursos. A análise dos dados de vendas e lucros por países foi essencial para embasar decisões estratégicas e impulsionar o sucesso do negócio em escala global.
